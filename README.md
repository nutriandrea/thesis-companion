# Thesis Companion

AI-powered companion for thesis writing and research management.

## Features

- **Smart Research** — PubMed/arXiv integration for literature discovery
- **Citation Management** — Automated bibliography generation
- **Writing Assistant** — AI-powered drafting and editing
- **Progress Tracking** — Milestone and deadline management

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

React · TypeScript · Vite · Supabase · PostgreSQL
